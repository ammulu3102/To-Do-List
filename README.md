## âš™ï¸ How It Works

1. User types a task in the input box.
2. Clicks the **Add** button to add the task.
3. The task is displayed in the todo list.
4. Each task has a **Delete** button to remove it.
5. Todos are saved in the browser using **IndexedDB**.
6. When the page is refreshed, saved todos are loaded again.
